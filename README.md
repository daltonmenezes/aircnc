# Aircnc
[WIP] Airbnb like (Air Coffee &amp; Code) to booking spots for developers.

>This project comes from Rocketseat's omnistack week crash course #9.

# Stack
- ReactJS (front-end)
- React Native (mobile)
- Node.js (back-end)
- Express (API)
- Socket.io (realtime)
- MongoDB (database)
- Mongoose (odm)
